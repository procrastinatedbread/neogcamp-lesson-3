Git vs Github

Git is basically a record keeping system but at a local level(in our case our PCs) whereas Github is a global record keeping system, the "hub" in Github represents the global part where it keeps track of what changes you made on what day and at what time. In easier words, Git is like our PCs which has some movies(programs of various projects) from various local sources and then we have something like Netflix which contains movies from all around the world.
