Git vs Github

Git is basically a book keeping mechanism but at a local level(in our case our PCs) whereas Github is a global book keeping mechanism, the hub represents the global thing where it keeps track of what changes you made on what day and what time. I have a very good example to clear out the difference( for once and for all) between these but it is explicit so read further at your own risk. So, git vs github let's replace the word git with porn so now it is porn vs pornhub porn here represents local explicit content you have on your device but the collection of porn from all the devices from all over the world at one place is called pornhub you can see how pornhub has all the details about when a particular thing was uploaded and when were the changes made, who commented what etc( you get the point).
